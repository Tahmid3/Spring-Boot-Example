About this Project

Page1 has a simple implementation of an Input - Data transport - Database - Data transport - Output system. I used a List as a "database" for this example.

This example shows you the basic architecture of a Full-Stack Web Application, consisting of the frontend (HTML - CSS - JS) and the backend (Java - "database")
